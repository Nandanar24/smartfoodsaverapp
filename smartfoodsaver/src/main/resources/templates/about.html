About SmartFoodSaver

SmartFoodSaver was created with one mission in mind: to help households save money and reduce food waste. By making it easier to track expiry dates, plan meals, and learn storage tips, families can take control of their kitchens while doing good for the environment.

Why it matters

  • Food waste costs households money and increases landfill emissions.
  • Small changes — better storage, smarter shopping, planned leftovers — add up.
  • Our tools make it simple: reminders, ideas, and clear storage guidance.

Project notes

  • This project was developed as part of an NCEA Digital Technologies project.
  • It continues to grow with your feedback and support.
  • Built using Spring Boot, Thymeleaf, and H2 Database for fast and reliable performance.

Database usage

SmartFoodSaver uses an H2 database for lightweight, in-memory or file-based storage — perfect for testing, development, and small-scale production environments.

  • User registration details (name, email, encrypted password) are securely stored.
  • Meal planner data, expiry reminders, and storage tips are linked to your account for a personalized experience.
  • Passwords are encrypted using BCrypt for enhanced security.
  • For future updates, the system can easily migrate to MySQL or PostgreSQL for long-term data persistence.
